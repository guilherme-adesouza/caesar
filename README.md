### **CAESAR**


#### **api** -> *http://localhost:5000*  
`cd server/`  
`npm run server`

**Available endpoints: `/api`**

These endpoints are ready to receive GET, POST, DELETE, and PUT calls.
- `/account`
- `/game`
- `/genre`
- `/platform`
- `/user`
- `/race`
- `/race-type`

Others availables endpoints.

- `/login`
- `/logout`
- `/verify-auth`
- `/generate-password`
- `/upload`


#### **client** -> *http://localhost:3000*  

`cd client/`  
`npm start`
